# 2026-04-29 Food Guide Phase 1 Implementation

## Goal

서울 구별 랜딩페이지에서 나들이 동선과 함께 볼 수 있는 먹거리 진입점을 만든다. 1차에서는 별도 API 키 없이 카카오맵 검색 링크를 사용하고, 맛집 순위나 실시간 영업 상태를 임의로 단정하지 않는다.

## Phase 1 Shipped

- `/food` 먹거리 허브 페이지 추가
- `/districts/[slug]/food` 구별 먹거리 페이지 추가
- 주요 10개 구 먹거리 검색 의도 데이터 추가
- 카카오 API 키 없이 동작하는 `map.kakao.com/link/search` 링크 생성
- 구별 상세 페이지, 헤더, 푸터, sitemap에 먹거리 진입점 연결
- 방문 전 영업시간, 휴무, 메뉴를 카카오맵 또는 공식 채널에서 재확인하라는 안내 추가

## Phase 2 Proposal

- 서울 열린데이터광장 일반음식점 인허가 데이터 기반으로 25개 구 식당 후보를 매일 수집
- 폐업, 영업 상태, 주소, 업태, 인허가일 기준으로 기본 필터링
- 구별 먹거리 페이지에 `최근 확인된 식당 후보` 섹션 추가
- 자동 수집 데이터와 수동 큐레이션 문구를 분리해서 신뢰성을 유지
- AdSense 관점에서는 얕은 맛집 리스트보다 구별 동선, 행사, 가족/비 오는 날/무료 코스와 연결된 실사용 페이지로 확장
