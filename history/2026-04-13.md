# 2026-04-13

## 작업 개요

서울시 행사/축제 데이터를 서울 열린데이터광장 기준으로 분리하고, `/events` 영역을 실제 운영 가능한 구조로 정리했습니다.

## 주요 작업

- 서울 열린데이터광장 `서울시 문화행사 정보`를 수집하는 파이프라인을 추가했습니다.
- 행사 데이터 저장 구조를 `public/data/events/index.json` + `public/data/events/items/*.json` 형태로 분리했습니다.
- 행사 목록 페이지 `/events`와 상세 페이지 `/events/[id]`를 새 데이터 구조로 연결했습니다.
- 홈 화면의 행사 섹션도 같은 이벤트 데이터 소스를 사용하도록 정리했습니다.
- 헤더의 `행사/축제` 링크를 고정 상세 페이지가 아니라 `/events` 목록으로 변경했습니다.
- GitHub Actions에서 서울시 행사 데이터를 매일 오전 7시 30분(KST) 기준으로 갱신하도록 연결했습니다.

## UX/정렬 개선

- 행사 목록에 구별 필터를 추가했습니다.
- 구 필터는 가로 스크롤이 아니라 전체가 한 번에 보이도록 정리했습니다.
- 행사 정렬 기준을 시작일 기준으로 맞췄습니다.
- 현재 날짜 기준으로 지난 행사는 목록에서 숨기고, 기간형 데이터는 오늘이 기간 안에 있으면 계속 보이도록 처리했습니다.

## 자동 반영 이력

- GitHub Actions 자동 업데이트 커밋으로 행사 데이터가 한 차례 더 갱신되었습니다.
- 신규 서울시 행사 데이터가 추가되면서 `public/data/events` 저장본도 최신 기준으로 다시 정렬되었습니다.
